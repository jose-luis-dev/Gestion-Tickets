# Sistema de Gestión de Tickets - Java

Proyecto desarrollado en **Java puro** con el objetivo de reforzar los fundamentos del lenguaje y la Programación Orientada a Objetos antes de avanzar a frameworks como Spring Boot.

Este sistema simula la gestión básica de tickets de soporte mediante consola.

---

## Funcionalidades actuales

- Crear tickets de soporte
- Listar tickets registrados
- Consultar ticket por ID
- Asignar prioridad al ticket (ALTA, MEDIA, BAJA)
- Gestionar estado del ticket (ABIERTO, EN_PROCESO, CERRADO)
- Mostrar información detallada de cada ticket
- Menú interactivo por consola

---

## Tecnologías y conceptos utilizados

- Java 17+
- Programación Orientada a Objetos (POO)
- Clases y objetos
- `ArrayList`
- `enum`
- Estructuras de control (`if`, `switch`, `for`)
- Entrada de datos por consola (`Scanner`)
- Manejo de fechas (`LocalDateTime`)
- Buenas prácticas básicas de encapsulamiento

---

## Objetivo del proyecto

Consolidar conocimientos de **Java puro** y lógica de programación antes de implementar soluciones con Spring Boot, evitando la dependencia de frameworks y comprendiendo a fondo el flujo de negocio.
